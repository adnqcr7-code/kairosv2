# Contributing to Kairos

We love contributions! Here's how to get started.

## Ways to Contribute

### 🐛 Report a Bug
1. Check if it's already reported
2. [Open an issue](https://github.com/adnqcr7-code/kairosv2/issues/new)
3. Include: reproduction steps, expected behavior, actual behavior

### 💡 Suggest a Feature
1. Check existing discussions
2. [Start a discussion](https://github.com/adnqcr7-code/kairosv2/discussions/new)
3. Explain the use case and benefit

### 👨‍💻 Contribute Code

**Setup development environment:**
```bash
git clone https://github.com/adnqcr7-code/kairosv2.git
cd kairosv2
npm install
npm run dev
```

**Run tests:**
```bash
npm run test
```

**Code standards:**
- ESLint checked: `npm run lint`
- Format with Prettier: `npm run format`
- Add tests for new features
- Update docs if needed

**Submit PR:**
1. Create feature branch: `git checkout -b feature/my-feature`
2. Make changes with tests
3. Commit: `git commit -m "feat: description"`
4. Push and open PR

### 📚 Improve Documentation
- Fix typos
- Add examples
- Improve clarity
- Translate to other languages

### ⭐ Spread the Word
- Star the repo
- Share on social media
- Write about Kairos
- Mention in articles

## Code of Conduct

- Be respectful and inclusive
- Assume good intent
- Focus on the issue, not the person
- Help others succeed

## Development Workflow

1. **Create issue** for bug/feature
2. **Discussion** on approach (for major features)
3. **Create branch** from main
4. **Implement** with tests
5. **Review** by maintainers
6. **Merge** when approved

## Need Help?

- Ask in [GitHub Discussions](https://github.com/adnqcr7-code/kairosv2/discussions)
- Check [documentation](./docs/)
- Review examples in `src/kairos/test/`

Thank you for helping make Kairos better! 🙏
